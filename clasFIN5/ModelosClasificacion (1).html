Clasificacin Binaria

Clasificacin por nivel de experiencia

1. La variable 'experience_level' clasifica a los empleados en:

  • EN: Entry-level (nivel de entrada)
  • MI: Mid-level (nivel medio)
  • SE: Seniorv (mas que medio)
  • EX: Executive (alta direccin)

2. Clasificacin binaria:

Grupo 0: EN y MI poca o mediana experiencia Grupo 1: SE y EX mucha experiencia

3. Para qu sirve?

Este modelo permite predecir si un perfil es de alta experiencia, basndose en variables como:

  • employment_type (tipo de contrato)
  • job_title o job_group (tipo de cargo)
  • company_size (tamao de empresa)
  • remote_ratio o is_remote (nivel de trabajo remoto)
  • company_location y employee_residence (ubicacin)
  • salary_in_usd

4. Aplicacin real:

  • Sistemas de recomendacin de empleo
  • Clasificacin automtica de candidatos por experiencia

3. Transformar variables categricas

Qu es astype('category')?

Es un mtodo de pandas que convierte una columna de tipo object (texto) o numrico en un tipo especial llamado Categorical.

Para qu sirve?

  1. Ahorra memoria

En vez de guardar cada string, guarda un nmero entero que representa la categora.

  1. Orden lgico o jerrquico (opcional)

Puedes definir un orden entre categoras si lo necesitas (por ejemplo: Junior < Mid < Senior < Exec).

  1. Facilita el preprocesamiento para Machine Learning

Muchos modelos no aceptan strings como entrada, y este paso permite aplicar transformaciones como: LabelEncoder, OneHotEncoder, Convertir fcilmente a variables dummies con pd.get_dummies()

  1. Mejora operaciones como groupby, sort, filter

Operaciones sobre datos categricos son ms rpidas y eficientes que sobre objetos de texto.

Para mejorar el modelo a continuacin se transforma y agrega una nueva variable al set de datos de caractersticas

Funcin group_job(title):

  1. Recibe como argumento un nombre de cargo (job_title).

  2. Convierte el texto a minsculas para evitar problemas con maysculas o minsculas.

  3. Usa una serie de condicionales if-elif para agrupar cargos similares en categoras generales:

  4. Si el ttulo contiene la palabra "scientist", lo clasifica como "scientist".

  5. Si contiene "engineer", lo agrupa como "engineer".

  6. Y as con "analyst", "manager", y "architect".

  7. Si no contiene ninguno de esos trminos, lo etiqueta como "other".

Aplicacin sobre el DataFrame:

  1. df['job_title'].apply(group_job) aplica esta funcin a cada fila de la columna job_title.

  2. El resultado se guarda en una nueva columna llamada job_group.

In[344]:
# Variable agrupada: grupo del cargo
def group_job(title):
    title = title.lower()
    if "scientist" in title:
        return "scientist"
    elif "engineer" in title:
        return "engineer"
    elif "analyst" in title:
        return "analyst"
    elif "manager" in title:
        return "manager"
    elif "architect" in title:
        return "architect"
    else:
        return "other"
# llenar columna job_group
df['job_group'] = df['job_title'].apply(group_job)
